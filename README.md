# 서울시 빅데이터 기반 열선 도로 우선 입지 선정/공급을 통한 겨울철 교통문제 해결
### (서울시 겨울철 도로 사고 위험도 분석을 통한 열선 도로 우선 입지 선정

🙋‍♀️팀원: 신상민, 김혜인, 전세환, 김지민, 김희수<br>
✏️ 분석 도구: Python, Excel, QGIS
